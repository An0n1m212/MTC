### Завдання 1: Ймовірнісний калькулятор
**Приклад:** Обчислення класичної ймовірності.
```python
n = 10 # Загальна кількість
m = 3  # Сприятливі події
if m <= n:
    print(f"P(A) = {m/n:.3f}")
```

### Завдання 3: Формула Бернуллі
**Приклад:**
```python
import math
n, k, p = 10, 2, 0.3
prob = math.comb(n, k) * (p**k) * ((1-p)**(n-k))
print(f"P = {prob:.4f}")
```
